🎨 Mint Asset Generation Instructions

  1. Right-click on each preview below and select "Save Image As..."
  2. Save the Launcher Icon as mint_logo.png (512x512)
  3. Save the Splash Screen as splash.png (1080x1920)
  4. Place files in assets/icon/ and assets/images/ respectively
  5. Run flutter pub run flutter_launcher_icons to generate adaptive icons

Alternative: Use an online SVG to PNG converter with the SVG files in the assets folder.
